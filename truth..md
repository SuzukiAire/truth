# ほんとうのわたし
## Part 0 オープニング
:::
召喚室。
プレイヤーの目の前には壊れて動かない転送機がある。
その近くの机の上には一冊の古びた本がある。本のタイトルは読めない。
本のすぐ隣にボタンがついた機械がある。
プレイヤーが本を機会にセットすると、機械の電源が入り、ボタンがオレンジ色に点灯する。
プレイヤーがボタンを押すと、しばらくののち、目の転送機が動き出す。
転送機が起動し、徐々に女王が現れる。
転送が完了ししばらくすると女王が目を覚ます。
女王、自分の身体を確認した後、あたりを見回す。
:::
女王    ……なんだここは？
:::
間。
:::
担当    お目覚めになりましたか、ご気分はいかがでしょうか。
:::
担当、プレイヤーの後ろから現れる。
:::
担当    ご気分はいかがでしょうか。
女王    なんだ……お前は？
担当    いろいろとご説明しなくてはいけないことはあるのですが……ひとまず窓の外をご覧ください。
:::
担当がパネルを操作する。
後ろの壁が開き、大都会が見える。
女王、驚き、二歩ほど後ろに下がる。
:::
女王    これは一体……！
担当    大丈夫、落ちませんよ、壁を透過させただけです。
:::
女王、しばらく窓の外を見る
:::
女王    ここはいったいどこだ……これほどの技術を持った国があったとは……。
担当    ここはあなたの国ですよ。女王様。
:::
女王、振り返る。
:::
女王    なに？
担当    ここはあなたの国です。但し、あなたが死んでから700年経った後の。
:::
部屋が暗くなり、音楽が流れる。担当と女王は消える。
部屋に置かれていたモニターにタイトルが映る。
『ほんとうのわたし』
:::
